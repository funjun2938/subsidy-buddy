# 정부 지원금·보조금 AI 매칭 서비스 — 구현 완료 기능

> 배포 URL: https://app-seven-inky-55.vercel.app
> 기술 스택: Next.js 16.2 · React 19 · Tailwind CSS · Gemini 2.0 Flash · Claude Sonnet (Fallback)

---

## 1. AI 맞춤형 지원사업 매칭 (핵심)

| 기능 | 설명 |
|------|------|
| 조건 기반 매칭 | 업종, 매출, 지역, 업력, 대표자 나이 5가지 조건으로 매칭 |
| AI 자동 분석 (파일) | 사업자등록증/서류 이미지 업로드 → Gemini Vision이 자동 분석 |
| AI 자동 분석 (텍스트) | 사업 설명 텍스트 입력 → AI가 조건 자동 추출 |
| 듀얼 입력 | "AI 자동 분석" 탭 + "직접 입력" 탭 선택 가능 |
| 드래그앤드롭 | 파일 드래그앤드롭 업로드 지원 |
| 매칭 스코어링 | 높음(emerald) / 보통(amber) / 낮음(gray) 3단계 점수 |
| D-day 표시 | 마감일까지 남은 일수 자동 계산 |

### 관련 페이지
- `/` — 랜딩 + 조건 입력
- `/results` — 매칭 결과 리스트
- `/grants/[id]` — 지원사업 상세 + AI 자격 분석

## 2. AI 신청서 생성 (BM 기능)

| 기능 | 설명 |
|------|------|
| 지원사업별 공식 양식 적용 | 예비창업패키지, 초기창업패키지 등 사업별 양식 자동 매칭 |
| 파일 업로드 | 사업자등록증, IR자료 등 첨부 → AI가 사업 정보 추출 |
| 필수 정보 체크리스트 | 지원사업별 필수 포함 항목 자동 표시 (12~14개 항목) |
| 자동 체크 기능 | 업로드/입력한 내용에서 필수 항목 포함 여부 AI가 자동 확인 (✅/❌) |
| 누락 항목 안내 | 미확인 항목에 대해 추가 작성 가이드 제공 |
| 사업계획서 생성 | AI가 공식 양식에 맞춘 완전한 사업계획서 초안 작성 |
| 복사/재생성 | 생성된 문서 클립보드 복사, 재생성 지원 |

### 지원 양식
- 예비창업패키지 (6개 섹션: 아이템 개요, 문제 인식, 실현 가능성, 성장 전략, 팀 구성, 자금 운용)
- 초기창업패키지 (6개 섹션: 아이템 현황, 시장성, 기술성, 사업화 전략, 팀 역량, 사업비 집행)
- 범용 정부 지원사업 (6개 섹션)

### 관련 페이지
- `/generate` — AI 신청서 생성

## 8. API 엔드포인트 (5개)

| Method | 경로 | 기능 |
|--------|------|------|
| POST | `/api/match` | 조건 매칭 → AI 분석 결과 |
| GET | `/api/grants?id=...` | 지원사업 상세 + AI 자격 분석 |
| POST | `/api/analyze-doc` | 파일/텍스트 → 사업 정보 추출 |
| POST | `/api/generate-doc` | 지원사업별 양식 사업계획서 생성 |
| POST | `/api/check-doc` | 필수 정보 체크리스트 자동 확인 |

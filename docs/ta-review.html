보조금 매칭 (GovGrant Match)

AI-Powered Government Subsidy Matching for Korean SMEs

Next.js 16 + React 19 Gemini 2.0 Flash + Claude Deployed on Vercel Due: Apr 9, 2026

1 Project Overview

Problem

Small business owners in Korea miss billions in government subsidies because information is fragmented across 100+ portals, eligibility rules are complex, and application formats differ per program.

Solution

An AI-powered platform that matches businesses to eligible subsidies using 5 key attributes, analyzes uploaded documents via Vision AI, and generates application drafts in official program formats.

Live Demo

https://app-seven-inky-55.vercel.app

2 Architecture & Features

Frontend (6 Pages)

  • / — Landing + condition input
  • /results — Matched grants list
  • /grants/[id] — Grant detail + AI analysis
  • /generate — AI application generator
  • /experts — Expert matching
  • /pricing — Subscription plans

API Routes (8 Endpoints)

  • POST /api/match — AI matching
  • GET /api/grants — Grant details
  • POST /api/analyze-doc — Document analysis
  • POST /api/generate-doc — Draft generation
  • POST /api/check-doc — Checklist verification
  • POST /api/revise-doc — Revision
  • GET /api/grant-redirect — External link
  • GET /api/experts — Expert list

AI Engine

  • Gemini 2.0 Flash — primary (matching, analysis, generation)
  • Gemini Vision — document image OCR
  • Claude Sonnet — fallback engine

Data Sources

  • 기업마당 공공API — real-time government data
  • Curated seed data — 30 programs (MVP fallback)
  • 1-hour cache with auto-expiry filtering

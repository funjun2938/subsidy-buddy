# Plan: 정부 지원금·보조금 AI 매칭 서비스

## 1. 개요

| 항목 | 내용 |
|------|------|
| 서비스명 | 보조금매칭AI (가칭) |
| 한줄 설명 | 소상공인·스타트업의 조건을 입력하면 AI가 신청 가능한 정부 지원금을 자동 매칭 |
| 기술 스택 | Next.js 15 (App Router) + Supabase + Claude API + 공공데이터 API |
| 초기 투자 | ~130만원 |
| 목표 | 3주 내 MVP 출시 → 3개월 내 월 매출 200만원 |

## 2. 문제 정의 (Pain Point)

- 매년 수천 개 지원사업이 산발적 공고 → 정보 과부하
- 내가 받을 수 있는 사업을 찾기 어려움
- 신청서 작성 난이도 높음
- 마감일 놓침 (알림 부재)
- 전문 컨설턴트 비용 부담 (건당 50~200만원)

## 3. 타겟 고객

| 단계 | 타겟 | 규모 |
|------|------|------|
| 1차 | 소상공인·자영업자 | 200만+ 사업체 |
| 2차 | 예비/초기 스타트업 | 연 10만+ 창업자 |
| 3차 | 중소기업 경영지원팀 | R&D 과제, 수출 지원 |

## 4. 수익 모델

| 수익원 | 가격 | 시작 시점 |
|--------|------|-----------|
| 기본 매칭 | 무료 (유저 확보) | MVP |
| 프리미엄 알림 + 상세 가이드 | 월 9,900원 | 6주차 |
| AI 신청서 작성 보조 | 건당 29,900원 | 6주차 |
| 전문가 대행 매칭 | 수수료 10~15% | 9주차 |

## 5. 기술 스택

- **프론트엔드**: Next.js 15 App Router + Tailwind CSS
- **백엔드/DB**: Supabase (PostgreSQL + Auth + Realtime)
- **AI**: Claude API (Sonnet) — 자격요건 분석 + 매칭 스코어링
- **데이터**: 공공데이터 API (정부24, 기업마당, 중기부)
- **알림**: 비즈엠 카카오 알림톡 API (추후)
- **결제**: 토스페이먼츠 (추후)
- **배포**: Vercel

## 6. MVP 범위 (3주)

### 포함
- 랜딩 페이지 + 조건 입력 폼
- AI 매칭 결과 페이지
- 지원사업 상세 페이지 + AI 분석
- 시드 데이터 (지원사업 30~50건)

### 미포함 (추후)
- 회원가입/로그인
- 카카오 알림톡
- 결제 시스템
- 전문가 매칭
- 관리자 대시보드

## 7. 핵심 차별화

1. **자기 서비스로 자기 자금 조달** — 이 서비스로 직접 창업지원금 매칭+신청 가능
2. **공공 API 무료 수집** — 데이터 비용 0원
3. **형(변리사) 네트워크** — R&D·특허 과제 전문 자문 + 전문가 풀
4. **AX 담당 경험** — 정부 사업 공고 구조 + 신청 프로세스 실무 이해

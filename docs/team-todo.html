Deadline: Apr 9, 2026 — 11:50 PM

Step 0: Before You Start (All Members)

  1. Jun(팀장)이 GitHub repo를 생성하고 공유할 때까지 대기
  2. Repo를 clone합니다:
    git clone https://github.com/[username]/govgrant-match.git cd govgrant-match
  3. 본인 이름의 branch를 만듭니다:
    git checkout -b feat/[your-task-name]
  4. 앱을 로컬에서 확인하고 싶으면 (선택):
    cd app npm install npm run dev # → http://localhost:3100 에서 확인
  5. 작업 완료 후 push → GitHub에서 PR 생성:
    git add . git commit -m "feat: [설명]" git push origin feat/[your-task-name] # → GitHub.com에서 "Compare & pull request" 버튼 클릭
Node.js 필요: 로컬에서 앱을 실행하려면 Node.js 18+ 설치 필요. 단, 코드 수정만 하고 push하는 것은 Node 없이도 가능합니다.
